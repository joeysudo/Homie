Icon files needed for the Homie extension:

1. icon16.png - 16x16 pixel icon
2. icon48.png - 48x48 pixel icon
3. icon128.png - 128x128 pixel icon
4. icon16_grey.png - 16x16 pixel greyscale icon
5. icon48_grey.png - 48x48 pixel greyscale icon
6. icon128_grey.png - 128x128 pixel greyscale icon

Please create these icons in PNG format with a house or real estate themed design.
The colored versions should use a blue/teal color scheme to match the extension's UI.
The greyscale versions should be the same design but in grayscale for inactive state.

You can use online tools like Canva, GIMP, or Photoshop to create these icons.
